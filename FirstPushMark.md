
# Git Push Test File

## Purpose
This is a test file to demonstrate using `git push` to upload local commits to a remote repository.

## What is Git Push?
**Git push** is a fundamental Git command that transfers your local repository commits to a remote repository (like GitHub, GitLab, or Bitbucket).

### Basic Syntax
```bash
git push <remote> <branch>
```

### Example
```bash
git push origin main
```

## Key Points
- 📤 Uploads your commits to the remote repository
- 🔄 Keeps your local and remote branches in sync
- ✅ Required before others can see your changes
- 🛡️ Always pull before pushing to avoid conflicts
